# Comparing Dummy Coding and Effect Coding for NCRF Estimation in MEG

This repository contains the code used to compare **Dummy Coding (DC)** and **Effect Coding (EC)** event encoding schemes for estimating **Neuro-Current Response Functions (NCRFs)** from discrete-trial MEG experiments.

The project investigates whether the choice of event encoding influences the estimated NCRFs, predictive performance, and resulting source localization.

---

## Dataset

This project uses the **NOD-MEG** dataset available on OpenNeuro:

**Dataset:** NOD-MEG (ds005810)

https://openneuro.org/datasets/ds005810

Download the complete dataset, including:

- MEG recordings
- Structural MRI (T1-weighted images)

The dataset follows the BIDS format and is publicly available through OpenNeuro. :contentReference[oaicite:0]{index=0}

## Pipeline

### 1. Download the dataset

Download the complete NOD-MEG dataset (including MRI) from OpenNeuro.

### 2. FreeSurfer reconstruction

Run FreeSurfer's `recon-all` for each subject to generate the anatomical surfaces.

### 3. Prepare subjects

- Define and save the anatomical fiducial points manually using the MNE coregistration GUI.

- Run

```text
Setup_subjects.py
```

to generate

- BEM solution
- Source space

for every subject. The generated files include:

- sub-XX-bem-sol.fif
- sub-XX-ImageNet0X-trans.fif
- sub-XX-vol-7-src.fif

### 4. Estimate NCRFs

Run

```text
estimate_ncrfs.py
```

to estimate NCRF models using both encoding schemes:

- NCRF-DC (Dummy Coding)
- NCRF-EC (Effect Coding)

### 5. Reproduce figures

All figures from the paper can be regenerated by running the scripts in the

```text
figures/
```

directory.
